# MathLab — Plataforma de Aulas Particulares de Matemática

Plataforma online para gerenciamento de alunos, turmas e sessões de aulas particulares de matemática.

## Módulos disponíveis

- Lógica Matemática (1º Ano EM)
- Razão e Proporção (ENEM)
- Números Racionais (7º Ano)
- Raciocínio Lógico (1º Ano EM)
- Números Inteiros (7º Ano)

## Tecnologias

- HTML/CSS/JavaScript puro
- Supabase (autenticação + banco de dados)
- GitHub Pages (hospedagem)
